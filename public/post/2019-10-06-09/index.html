miniDLNA配置

安装

sudo dnf install minidlna
配置文件位置/etc/minidlna.conf

sudo vim /etc/minidlna.conf

media_dir=/opt   #资源文件路径

启动服务sudo systemctl start minidlna
添加为开机启动sudo systemctl enable minidlna
取消开机启动sudo systemctl disable minidlna
